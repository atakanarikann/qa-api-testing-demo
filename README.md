# API Testing Demo — ReqRes API

This project contains manual and Postman API testing for the public ReqRes REST API.

## Endpoints Covered
- GET /users?page=1  
- GET /users/{id}  
- POST /users  
- PUT /users/{id}  
- DELETE /users/{id}  

## Artifacts
- **API_TestCases.md** — 20+ test cases  
- **API_BugReports.md** — Found issues  
- **Postman_Collection.json** — Ready-to-import test suite  

## Tools Used
- Postman  
- JSON Validator  
- GitHub Documentation
